### 微信小程序 积分兑换

#### 项目介绍
主要是通过做任务来获取积分，积分可以兑换商品，也可用来玩游戏；通过玩游戏既可能获取奖品或积分也可能会消耗积分；由于出自己构思，所有后面的做任务功能和兑换商品的功能及其详情页都没做
